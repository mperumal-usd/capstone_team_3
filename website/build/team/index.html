About the Capstone​

This project is the culminating work for AAI-590: Applied AI Capstone at the University of San Diego. The course challenges students to apply the full machine learning lifecycle — problem definition, data engineering, model development, evaluation, and communication — to a real-world domain problem.

Capstone focus: Music Information Retrieval (MIR) — specifically the problem of scalable melody similarity search using modern audio foundation models.
